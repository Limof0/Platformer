# Платформер "Прыгун"
(В данный момент стадия разработки)

Простая игра-платформер с 10 уровнями, созданная на Python с использованием библиотеки Pygame.

## Особенности игры
- 10 уровней с постепенным усложнением
- Разные типы платформ: обычные, движущиеся, прыгучие
- Враги, которые патрулируют территорию
- Система жизней
- Камера, следующая за игроком

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Установите необходимые библиотеки: pip install -r requirements.txt
3. Запустите игру: python main.py

## Управление
- **A / D или ← / →** - движение влево/вправо
- **SPACE / W / ↑** - прыжок
- **R** - перезапуск уровня
- **N** - следующий уровень (после завершения текущего)
- **P** - пауза
- **ESC** - выход из игры

## Цель игры
Пройти все 10 уровней, прыгая по платформам и избегая врагов. Дойдите до желтой цели "FINISH" на каждом уровне.

## Структура проекта
- `main.py` - главный файл для запуска игры
- `game.py` - основные классы и логика игры
- `levels.py` - данные об уровнях
- `requirements.txt` - зависимости проекта

## Для разработчиков
Игра создана как пример простого платформера. Вы можете легко:
- Добавить новые уровни в `levels.py`
- Изменить параметры игрока (скорость, силу прыжка)
- Добавить новые типы платформ или врагов
- Включить звуки и музыку
